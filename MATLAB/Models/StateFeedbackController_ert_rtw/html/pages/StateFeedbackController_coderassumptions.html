'StateFeedbackController' のコード生成の前提

チェック可能なコード生成の仮定と、選択したターゲット環境で予測される結果。 関連ドキュメンテーション。

ARM Compatible->ARM Cortex-M ターゲット ハードウェア用の C 言語構成

BitPerChar 8

BitPerLongLong 64

BitPerDouble 64

BitPerPtrDiffT 32

エンディアン LittleEndian
符号付き整数の右シフトは算術シフト True
符号付き整数の除算の丸め Zero

C 言語標準

ゼロ初期化コードがモデル 'StateFeedbackController' 用に最適化されました。

整数のグローバル変数の初期値がゼロ True
動的に割り当てられる int 配列の各要素の初期値がゼロ True

コード ジェネレーターの仮定が正しくない場合は、モデル コンフィギュレーションの対応する最適化パラメーターを無効にするか、仮定に合わせてターゲット環境構成を変更してください。

浮動小数点数

コード生成は、浮動小数点数をサポートするように構成されています。ターゲット環境では、非正規数の計算が行われる可能性があります。

計算された非正規値が flush-to-zero (FTZ) False
入力の非正規値が flush-to-zero (DAZ) False

コード ジェネレーターの仮定が正しくない場合、非正規数が原因でモデルと生成コードのシミュレーション結果が一致しないことがあります。
